# Opize Component

`Opize Component`는 HTML 태그 기반이 아닌 컴포넌트입니다.

## Overview
|컴포넌트|설명|
|-|-|
|`<ActionBox>`|클릭했을 때 오버레이 박스를 표시하는 `<Button>`입니다.|
|`<ActionList>`|클랙했을 때 오버레이 버튼 리스트를 표시하는 `<Button>`입니다.|
|`<ActionMenu>`|액션 버튼 리스트입니다.|
|`<StatusBadge>`|상태를 간략히 표시하기 위한 뱃지입니다.|
|`<Box>`|Header와 Footer을 가질 수 있는 박스입니다.|
|`<ButtonGroup>`|`<Button>` 여러 개를 묶을 수 있는 컴포넌트입니다.|
|`<CodeBlock>`|여러 줄을 코드를 표시할 수 있는 컴포넌트입니다.|
|`<Footer>`|페이지 가장 하단에 표시하는 푸터입니다.|
|`<Header>`|페이지 가장 상단에 표시하는 헤더입니다.|
|`<ItemsTable>`|표와 비슷한 역할을 하는 컴포넌트입니다.|
|`<PageHead>`|페이지 상단 제목 등을 표시할 수 있는 컴포넌트입니다.|
|`<PageLayout>`|일반적인 페이지의 레이아웃을 잡아줍니다.|
|`<Spacer>`|빈 공간을 생성합니다.|
|`<Spinner>`|주로 로딩 상태를 표시하기 위해 사용되는 아이콘입니다.|
|`<TabNav>`|탭 네비게이션입니다.|
|`<ToolTip>`|마우스를 올렸을 때 툴팁을 표시하는 컴포넌트입니다.|
