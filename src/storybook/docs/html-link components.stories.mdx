# HTML-Like Component

HTML-Like Component는 `HTML` 기본 태그를 Opize의 방식으로 풀어낸 컴포넌트입니다.

대부분 `ref`를 사용할 수 있으며, 일반적인 html 태그처럼 사용할 수 있습니다.

## Overview
|컴포넌트|관련 HTML|
|-|-|
|`<Divider>`|`<hr>`|
|`<Avatar>`|`<img>`|
|`<Button>`|`<button>`|
|`<Code>`|`<code>`|
|`<Autocomplete>`|`<input type='text'> with <datalist>`|
|`<Checkbox>`|`<input type='checkbox'>`|
|`<Datetime>`|`<input type='date'>`|
|`<Radio>`|`<input type='radio'>`|
|`<Select>`|`<input type='select'>`|
|`<Switch>`|`<input type='checkbox'>`|
|`<TextArea>`|`<textarea>`|
|`<TextField>`|`<input type='text'>`|
|`<Flex>`|`<div style="display: flex">`|
|`<Link>`|`<a>`|
|`<Span>`|`<span>`|
|`<Text>`|`<p>`|
|`<Title>`|`<j1>, <h2>, <h3>`|
