### v0.12.0
1. ✨❗`<ItemsTable>`의 구조가 크게 변경되었어요.
    > 💬 기존에는 `data prop`을 통해 생성하였으나 현재는 `children`을 통해 생성하도록 변경되었어요.
2. ✨❗`<Button>`을 비롯한 몇몇 컴포넌트에 `as` 속성을 추가했어요.
    > 💬 이제 `asProps`가 아닌 자체 props로 추가적인 props을 추가할 수 있어요. ex. `<A as={Link} to='...'>A</A>`
3. ✨❗`react-router-dom` 의존성을 삭제했어요. 이제 기본적으로 `<a>`태그가 사용되며, 필요한 경우 `as prop`을 통해 `<Link>`를 사용할 수 있어요.
    > 💬 다만 Link를 사용해야 하는 모든 상황에서 `as={Link} to='...'`을 하는 것이 그리 편해보이지 않는 것 같아 이를 해결할 방법을 떠올리고 있어요.<br />
    > 💬 아직 의존성 최적화를 진행하지 않아 사용만 안할 뿐 라이브러리 자체는 `package.json`에 남아있어요. 이 부분은 이후 마무리 단계에서 해결할 예정입니다.
4. ✨ displayName이 필요한 컴포넌트에 displayName을 추가했어요.
5. 📝 스토리북에 subcomponents로 관련 컴포넌트들을 추가했어요. 이 정보들은 각 컴포넌트 Docs에서 확인할 수 있어요.
    > 💬 현재 displayName이 설정된 subcomponent는 스토리북 Docs이 불러오지 못하는 버그가 있는 것 같아요. 죄송하지만 subcomponent의 props은 예제 코드를 통해 파악해주세요. 이 부분은 추후 해결해보겠습니다.
