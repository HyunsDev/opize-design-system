# Work In Process

Opize Design System은 현재 **개발중인** 상태입니다. 아직은 사용을 권장하지 않습니다.

### 개발중인 상태로 인한 영향
0. 컴포넌트의 `Porps`가 예고 없이 변경될 수 있습니다.
1. 해결되지 못한 버그로 인해 치명적인 문제가 발생할 수 있습니다.
2. 코드 최적화가 이루어지지 않아 퍼포먼스에 영향을 미칠 수 있습니다.
3. 문서가 아직 작성되지 않을 수 있습니다.
4. 중요한 작동 구조가 예고 없이 변경될 수 있습니다.
5. 기타 등등
