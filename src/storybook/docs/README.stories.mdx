# Opize Design System

Opize Design System은 Opize를 위한 새로운 디자인 시스템입니다.

## WIP

Opize Design System은 현재 개발중인 프로젝트로, 아직은 이 라이브러리를 사용하는 것을 추천하지 않습니다.


## Install

- npm: `npm install opize-design-system`
- yarn: `yarn add opize-design-system`


## Use

Opize Design System은 내부적으로 Context를 사용합니다. 따라서 Opize Design System을 사용하기 위해서는 상위 레벨에 `OpizeWrapper` 컴포넌트를 추가해야 합니다.

```jsx
<OpizeWrapper>
    { children }
</OpizeWrapper>
```

컴포넌트는 다음과 같이 불러와 사용할 수 있습니다.

```jsx
import { Button, TextField } from 'opize-design-system';
```
