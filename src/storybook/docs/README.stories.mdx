## Overview
Opize Design System은 디자인에 대한 고민 대신, 아이디어를 현실로 만들도록 도와주는 컴포넌트 라이브러리이자 디자인 시스템입니다.

Opize Design System는 다음과 같은 수식어를 가지고 만들어집니다.
* **일관적인**: 다양한 형태의 독특한 디자인보단 일관적이고 통일된 디자인을 향합니다.
* **결합하기 쉬운**: 다양한 라이브러리의 사용을 고려합니다.
* **직관적인**: 아이디어 로직을 쉽게 표현할 수 있도록 직관적으로 설계합니다.
* **기능적인**: 필요한 기능을 미리 포함하여 빠르게 사용하도록 돕습니다.
* **친절한**: 주석과 문서를 친절하고 이해하기 쉽게 작성합니다.

## ⚠️ Work In Process - 개발중인 프로젝트
Opize Design System은 현재 개발중인 프로젝트로, 아직은 이 라이브러리를 사용하는 것을 추천하지 않습니다.

0. 컴포넌트의 `Props`가 예고 없이 변경될 수 있습니다.
1. 해결되지 못한 버그로 인해 치명적인 문제가 발생할 수 있습니다.
2. 코드 최적화가 이루어지지 않아 퍼포먼스에 영향을 미칠 수 있습니다.
3. 작성&보완되지 않은 문서가 존재할 수 있습니다.
4. 중요한 작동 구조가 예고 없이 변경될 수 있습니다.
