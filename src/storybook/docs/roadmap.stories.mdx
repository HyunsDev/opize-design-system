# Roadmap
* 키 조합(단축키) 추가
    > 디자인쪽이랑은 멀긴 하지만 UX 관점에서 꽤나 괜찮은 기능인 것 같습니다.
* Placeholder와 label 통합
    > 구글처럼 Placeholder에 있던 글자가 -> label로 가는 식으로 구현할 것 같습니다.
* 커스텀 Theme 지원
    > 다크 모드의 연장선이라고 보면 될 것 같습니다. 
* 모바일 지원
    > 이미 모바일 환경에서 레이아웃이 깨지지 않고 볼 수 있도록 마련은 해놓았으나, 아직 모바일에서 사용하기에 **편하지**는 않습니다. 이 부분을 개선할 생각입니다.
* Props 정리
    > 현재 ODS의 Props는 너무 난잡한 상태입니다. 이를 정리하려고 합니다.
