# Colors

## Usage

```jsx
import { cv } from 'opize-design-system'
import { styled } from 'styled-components'

// cv.gray100 === 'var(--gray100)'
const Text = styled.p`
    // 방법 1
    background: ${cv.gray100};

    // 방법 2
    background: ${({theme}) => theme.cv.gray100};
`
...
<Text>Text</Text>
```

> ### 💬 Comment
>
> 과거 ODS는 다크모드를 지원하였으나, 개발 자원 상의 문제 현재는 지원하지 않습니다.
